# SELHONO - Interior Design Landing Page

Одностраничный сайт для студии дизайна интерьера, разработанный по свободному дизайн-макету. Проект посвящён практике адаптивной верстки и созданию интерактивного пользовательского интерфейса.

##  Технологии

* HTML5
* CSS3 / SCSS
* JavaScript (Vanilla)
* jQuery
* Owl Carousel
* jQuery Mask Plugin

## Описание проекта

В рамках проекта был разработан лендинг, посвящённый услугам студии дизайна интерьера.
Основной упор был сделан на:

* адаптивную верстку по дизайн-макету;
* реализацию интерактивных элементов интерфейса;
* работу с DOM и обработкой пользовательских событий;
* подключение и настройку сторонних JavaScript-библиотек.

## Реализованный функционал

* Адаптивная верстка страниц
* Плавая прокрутка между секциями (Smooth Scroll)
* Несколько независимых каруселей с использованием Owl Carousel
* Кастомные кнопки управления слайдерами
* Форма обратной связи
* Валидация формы
* Маска ввода номера телефона
* Очистка формы после успешной отправки
* Интерактивные элементы интерфейса

## Особенности проекта

* Работа с DOM на JavaScript
* Использование jQuery для реализации интерактивности
* Интеграция сторонних библиотек (Owl Carousel, jQuery Mask Plugin)
* Создание отзывчивого пользовательского интерфейса
* Практика разработки лендингов по готовым дизайн-макетам

## Примечание

Проект выполнен в учебно-практических целях для закрепления навыков HTML, CSS и JavaScript, а также работы с библиотеками jQuery и создания современных адаптивных интерфейсов.
